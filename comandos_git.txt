git inic. 		-> cria um repo git na pasta atual
git status 		-> mostra o status do trabalho 
git add nome_arquivo 	-> adiciona arquivo para ser comitado
git commit -m "msg" 	-> faz commit que estava para ser comitado 
git log 		-> mostra todos o histórico de commit

>>FLUXO DE TRABALHO<<
1. criar o diretório do projeto
2. iniciar o git dentor do diretório criado (git init)
3. edita ou cria arquivos 
4. git status
5. git add <nome arquivo>
6. git status
7. git commit -m 'Mensagem'
8. git status
9. volta para o passo 3
